Header and footer toolbars

A toolbar wraps by default. Inside a header or footer it uses flex-wrap: nowrap and overflows when its actions exceed the available width.

Project settings

<header>
  <h3>Project settings</h3>
  <nav data-toolbar aria-label="Project actions">
    <button type="button">Share</button>
    <button type="button">Export</button>
  </nav>
</header>
<footer>
  <nav data-toolbar aria-label="Form actions">
    <button type="button">Cancel</button>
    <button type="submit">Save changes</button>
  </nav>
</footer>

API

Toolbar elements, attributes, tokens and states
NameTypeDefaultDescription
navElementRequiredProvides the nav[data-toolbar] selector host.
data-toolbarAttributeRequiredApplies the flexible toolbar layout to a nav.
--spacing-2 / --spacing-3TokenThemeControl the gap and default toolbar padding.
--border-size-1 / --default-lightTokenThemeControl the toolbar divider.
header or footer ancestorStateWrappingRemoves padding and divider and prevents wrapping.
