Table API

Responsive table attributes
NameTypeDefaultDescription
captionTable elementRequired for these examplesNames the table for readers and assistive technology.
scope=colth attributeColumn headerAssociates each heading with the cells in its column.
data-labeltd attributeMatching column headingProvides the visible mobile label generated by the responsive table CSS.
data-tableTable attributeUncolored tableAccepts primary, accent, success, info, warning, danger or default.

Accessibility and common errors

  • Keep a descriptive caption and use scope="col"on every column header.
  • Make each data-label match its column heading exactly so the compact layout preserves the same meaning.
  • Do not use a table only for visual alignment. Use it when rows and columns describe real data relationships.
  • The current responsive CSS hides thead visually. The semantic headers remain in the markup, while data labels provide the compact visual headings.
